# dcp

[Daily coding problem](https://www.dailycodingproblem.com/) solutions, organized by `<projectRoot>/year/month/day.ext`.

## Usage

To run, use the respective language REPL.

### Haskell
```shell
$ ghci
Prelude> :l 2018/11/25
*Main> solve ...
*Main> :q
$
```

### TypeScript
```shell
$ ts-node
> import { solve } from './2018/11/25'
> solve(...)
> .exit
$
```
